## 🛠️ Installation

Clone the repository

```bash
git clone https://github.com/devgoswami04/Portfolio.git
```

Move into the project

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
